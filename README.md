# How to make own bio website

1. [index.html](https://github.com/fuckautism/autism/blob/main/index.html) - change main texts
2. [assets](https://github.com/fuckautism/autism/tree/main/assets) - change here back, music, pfp, favico (default name only!!!)
3. [javascript](https://github.com/fuckautism/autism/tree/main/assets/js) - javascripts
3.1 [script.js](https://github.com/fuckautism/autism/blob/main/assets/js/script.js) - here you can change ASCII art and info
3.2 [title.js](https://github.com/fuckautism/autism/blob/main/assets/js/title.js) - dynamic title (longer - more in var titles)
